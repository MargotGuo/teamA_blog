CSS Basics

Nov 22, 2019 in Wuhan, release

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML.

Positioning layouts in CSS was once a very daunting task, and hacks like using tables for the entire layout were quite common. Over the years, the demand for better layout tools has led to steadily better support and techniques.
